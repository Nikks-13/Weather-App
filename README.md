
## Simple Weather App Build Using React-Native
## Run My project Locally

Clone the project

```bash
  git clone https://github.com/Nikks-13/Weather-App
```
Go to the project directory Open it and Install dependencies

```bash
 npm install
```

Start the server

```bash
 npm Start 
```
 on android
```bash
 press a
```
 on ios
```bash
 press i
```
